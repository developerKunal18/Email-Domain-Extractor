# Email Domain Extractor

A Python script that extracts the domain name from an email address.

## Description
This project takes an email address as input and displays the domain part after the `@` symbol.

## Features
- Email parsing
- Simple validation
- Beginner-friendly logic

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python domain_extractor.py

4. Enter an email address when prompted

## Example

Input:
example.user@gmail.com

Output:
gmail.com
